## 先说一个关键事实：两家的CN2 GIA，不是同一个东西

CN2 GIA这个名字，严格来说指的是中国电信的精品骨干网（AS4809），走这条线路的数据包会绕过普通163骨干网的拥堵节点，延迟更低、晚高峰更稳定。

但——

搬瓦工的CN2 GIA是**三网全程CN2 GIA**：电信、联通、移动，去程和回程都走优化线路。这是它真正的核心竞争力，你随便什么时候测试，路由追踪里都能看到稳定的59.43节点。

HostDare CN2 GIA则只有电信是CN2 GIA，移动和联通走各自的直连，和CN2 GIA没有关系。 再往深一点说，HostDare回程三网确实走了优化通道（电信CN2 GIA、联通AS9929、移动CMIN2），但去程是三网普通直连，这也是晚高峰延迟会出现波动的根本原因。

这个差距，决定了后续所有对比的基调。
